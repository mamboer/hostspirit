HostSpirit and Node-Webkit

A node-webkit app built in less than 1 day! (levinhuang at 2013.02.22)

What is Node-Webkit
  1. App runtime based on Chromium and node.js
  2. Lets you call Node.js modules directly from DOM!
  3. Enables a new way of building native applications with web technologies!
  4. Created and developed in Intel Open Source Technology Center
  5. Introduction to node-webkit (slides)
What is HostSpirit
  1. A Node-Webkit application
  2. Open source - A tutorial for learning Node-Webkit
  3. A hosts file management tool for Windows users!
  4. Lets you create multiple host settings
  5. Support enabling multiple host settings at the same time

How to use?*

* HostSpirit

1, Download and Extract

  • Download HostSpirit
  • Extract it to somewhat folder
  • E.G: D:\Program Files\HostSpirit\

2, Click hostspirit.exe !

1，Auto.Backup

  • HostSpirit will backup your hosts into a file,
  • When you run hostspirit at the first time,
  • The backup file is named 'hosts.hsbackup'
  • You will see the hsbackup in the settings at the next time you run it!

2，Add a new setting file

  1. Click the 'New' button
  2. Entry the Popup form
  3. Click 'Save' Button!

3，Switch host settings

  1. Select one or more settings
  2. Click 'Save' Button!
  3. Remember to click the 'Save' button!!!

4，Edit host setting

  1. Hover a setting
  2. Click the 'Edit' icon!

That's it!
